# VCC Time Wizard

A Raycast extension that copies the current time across multiple timezones to your clipboard.

## Commands

- **Copy Team Times** - Copy current time for all configured timezones
- **Convert Time** - Convert a specific time (HH:MM) to all timezones
- **Convert Date & Time** - Convert a specific date and time (DD/MM HH:MM) to all timezones

## Configuration

Configure up to 3 timezones with custom emojis in the extension preferences.
